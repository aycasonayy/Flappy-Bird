# Flappy Bird – 2D Oyun Prototipi

Bu proje, Unity ve C# kullanılarak geliştirilmiş, Flappy Bird oyunundan esinlenen bir 2D oyun prototipidir.

## Esinlenme
Proje, YouTube üzerindeki eğitim videoları takip edilerek geliştirilmiştir ve üzerine eklemeler yapılmıştır.  
Videolar, Unity 2D oyun geliştirme süreci, temel oyun mekaniği ve C# scriptleme mantığını öğrenmek amacıyla kullanılmıştır.  
Oyun, öğrenme ve portföy amaçlı olarak yeniden oluşturulmuş ve kişisel olarak uyarlanmıştır.

## Oynanış
- Oyuncu bir kuşu kontrol eder  
- Engeller arasından geçerek skor kazanılır  
- Çarpışma durumunda oyun sona erer  

## Kullanılan Teknolojiler
- Unity  
- C#  
- 2D Oyun Geliştirme  

## Amaç
Bu proje, Unity ile 2D oyun geliştirme becerilerimi geliştirmek ve oyun mekaniği üzerine pratik yapmak amacıyla oluşturulmuştur.
